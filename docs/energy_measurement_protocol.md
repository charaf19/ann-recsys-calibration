# CPU energy measurement protocol

The optional energy analysis measures CPU package energy directly through Linux RAPL (`/sys/class/powercap`) when readable. If direct counters are unavailable (e.g. on Windows), it falls back to timing, CPU utilization, and RSS while marking `direct_energy_available=false`; it never estimates or fabricates joules.

The measurement output is `results/analyses/energy/energy_measurement_all.csv` (written by `src/run_energy_measurement.py`); the presentation table is regenerated by `src/tables_paper.py`. The schema contains no GPU-energy columns and the analysis has no GPU dependency.
